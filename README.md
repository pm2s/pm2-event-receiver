Launches RabbitMQ server on `5672` port with management tool on `15672` port

## Installation
```
git clone https://github.com/pm2s/vagrant-rabbitmq.git
cd vagrant-rabbitmq
vagrant up
```

## Usage
* Go to [localhost:15672](http://localhost:15672/)
* Login with credentials
```
user: 'test',
password: 'test'
```
